## Problem

The project has grown from a script into a multi-tool profile manager. The core
behavior is testable, but configuration, installation, and repeated daily use
still need polish so the tool feels predictable rather than experimental.

## Desired End State

Users should be able to install, inspect config, tune quota behavior, view clean
tables on normal terminals, and run common workflows without remembering hidden
environment variables.
