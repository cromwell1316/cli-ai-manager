## Context

Windows and WSL support now spans installer, helper, credential conversion, and
sync behavior. Users need one repository-native runbook.

## Problem

The external AGY multi-account notes should not be required to complete normal
setup, recovery, or troubleshooting.

## Strategy

Consolidate installation, profile setup, sync, recovery, diagnostics, and known
limitations into maintained project documentation.

## Expected Result

Users can install and operate the manager on Windows and WSL from repository
docs alone.
